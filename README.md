# AbeLab AutoBuilder

## 概要
AbeLab AutoBuilder は、作りたい内容を日本語で伝えるだけで、パソコンが自動でツール（仕組み）を作成し、動作する状態まで完成させることを目指す「自動ツール作成システム」です。

利用者が要望を文章で入力すると、必要な機能を整理して設計し、ツールを作成し、動作確認までを自動で行ったうえで、完成品として出力することを目標とします。

## 目的
- 専門的な開発作業や手作業の負担を減らす
- 業務改善（効率化・ミス削減・停止リスク低減）を速く確実に実現する
- 1回限りの作成ではなく、同様のツールを繰り返し生成できる仕組みを作る

## 想定する使い方（例）
- 「CSVを指定のExcel形式に変換したい」
- 「PDFをテキスト化してフォルダに保存したい」
- 「見積→請求書→PDF→メール下書きまで自動化したい」

## 開発方針（段階的に実現）

### Phase 1：MVP（最小実用）
- 依頼文（要望）を受け取る
- 仕様（入出力・例外・ログ）を整理する
- ひな形からツールを生成し、実行できる形で出力する

### Phase 2：自動テストと自己修正
- 実行結果（成功／失敗）を取得してログ化
- 失敗時に原因を整理し、再生成・修正を繰り返す

### Phase 3：量産と配布
- テンプレート拡張（Excel／CSV／RPA／Web）
- 配布用パッケージ化
- 説明書生成、販売ページ文の生成
